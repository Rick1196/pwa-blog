FAQ's

How to login

  1. First you need to click on the button marked with the number "1", this will display the next page
  2. Next you need to provide the email and password you registered, the login button marked with the number "3" will be enable until you provide this two fields
  3. If you dont have an account, please click in the cancel button marked with the number "3" and then click the Register button(in the next part we will explain how to create an account)

How to Signup

  1. First you need to click on the button marked with the number "2", this will display the next page
  2. Next you need to provide the email,password and an username to register, the signup button marked with the number "4" will be enable until you provide this three fields
  3. If the email you provide is already registered, you will need to provide another one

How to publish a new post

  1. First you need to be logged in an then go to the "Create a new post" page
  2. Then you will need to provide the next fields:
    1. The post title, is a required field, you cannot post it without a Title
    2. A link to the image you want to be displayed in the list post's, you need to provide this link
    3. A link to the image banner for yout post, you need to provide this link
    4. A link to the video you want to show on your post, it's optional
    5. A link to the audio you want to play on your post, it's optional
    6. In this rich text editor you can redact all your post content with the format you want, also you can add some links to youtube videos and add some code snippets
    7. The momment you complete all the required fields, the "Save post" button will enable, you just need to click on it
